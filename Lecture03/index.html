Git

The most popular distibuted version control system

Installation 


$ sudo apt-get install git
# Or something similar depend on your distro
$ git --version
# You should see your git version

Why we need git?

Basic Usage


$ mkdir my_project
$ cd $_
$ git init                        # initial a git repo
$ echo "Hello world!" > README.md
$ git add README.md               # add README.md into WD
$ git commit -m 'initial commit'  # commit to repo
Note: 後面也是都用 WD 來當作 working directory 的縮寫

basic git command

下面的 command 都是接在 git 後面
  • add: 將後面的檔案加入 WD
  • commit: 將目前的 WD commit
  • status: 觀看目前 WD 的狀態
  • branch: 觀看 branch 的狀態

Practice Time

  • 新增一個名為 my_project 資料夾並初始化一個 git repo
  • 新增一個檔案 README.md ，裡面雖然寫一些內容
  • 用 status 觀看目前 WD 的狀態
  • 將 README.md 加入 WD 後再觀看一次狀態
  • 確認無誤後將目前的狀態 commit
Tip: 可以用 git add . 來加入所有有修改的檔案進 WD 喔！

Git Workflow

.gitconfig

An example of .gitconfig 

[user]
    name = YourName
    email = YourEmail
[alias]
    ci = commit
    st = status -sb
    # ... something more ... (you should remove this line)
[push]
    default = tracking
[branch]
    autosetuprebase = always
[color]
    ui = true

Practice: create your config file in ~

Work with remote repo

  • clone <URL> : 將URL 的 repo 複製一份到 local 端
  • push: 將目前 branch 的 commit 上傳到 remote 的 tracking branch
  • pull: 下載 remote tracking branch 的 commit 到 local
