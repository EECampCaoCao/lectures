前端技術

HTML

決定網頁的架構！ ex: 標題、框架、區塊…

CSS

決定網頁元素的style! ex: 顏色、大小、對齊…

JS

讓網頁動起來！ ex: 與使用者互動、動話、伺服器溝通…

HTML 語法

標籤語法，主要有兩種標籤:

  • 成對標籤: TagName ... /TagName
  • 單獨標籤: TagName

Dom tree 

<html>
  <head>
    <title> HTML5 is cool </title>
    <meta name="author" content="NTUgEEk">
    <meta charset="UTF-8">
  </head>
  <body>
    <h1> Heading 1 </h1>
    <p> Paragraph 1 : A Link <a href="google.com">Here</a> </p>
    <ul> <li> List 1 </li> <li> List 2 </li> <li> List 3 </li> </ul>
  </body>
</html>
